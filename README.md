# Remix Starter

## Features

- [TRPC](https://trpc.io)
- TailwindCSS and shadcn-ui

## Development

Run the dev server:

```bash
pnpm dev
```

## Deployment

First, build your app for production:

```bash
pnpm build
```

Then run the app in production mode:

```bash
pnpm start
```

Now you'll need to pick a host to deploy it to.
